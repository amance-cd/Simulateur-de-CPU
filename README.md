Projet mené dans le cadre de l'UE LU2IN006 - Structure De Données

Par Rémi FUNES & Amance CELIER DENNERY

Récapitulatif du sujet :
Le projet consiste à concevoir un simulateur de CPU en C, incluant la gestion de la mémoire, l'exécution d'instructions, et la mise en œuvre d'un parser pour un langage assembleur simplifié, avec un accent sur la gestion dynamique des segments et des modes d'adressage. 
Il est composé de 8 exercices, chacun permettant de développer une partie du simulateur.

COMPILATION : make all

Exécution : ./main code.asm
